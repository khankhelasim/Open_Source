# Lab 4: File and Directory Permissions (RHCSA Rapid Track)


## Overview
This lab focuses on **Linux file permissions, ownership, and Access Control Lists (ACLs)**. You will practice controlling access using `chmod`, changing ownership with `chown`, and applying **ACLs** with `setfacl`â€”all core **RHCSA exam objectives**.


## Objectives
- Understand read, write, and execute permissions
- Modify permissions using numeric and symbolic notation

- Implement and verify ACLs
- Troubleshoot permission-related issues


## Environment
- RHEL / CentOS-based Linux
- Root or sudo access
- ACL utilities installed
