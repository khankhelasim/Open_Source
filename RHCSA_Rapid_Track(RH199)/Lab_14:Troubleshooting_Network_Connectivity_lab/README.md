## Lab Overview

This lab focuses on identifying, diagnosing, and resolving common network connectivity issues in Linux environments. Students will use standard networking tools, NetworkManager (nmcli), and firewalld to systematically troubleshoot and fix network-related problems.

## Objectives

By the end of this lab, students will be able to:

* Test network connectivity using ping, traceroute, and nslookup
* Interpret network diagnostic command output
* View and manage network configurations using nmcli
* Configure and troubleshoot firewall rules using firewalld
* Apply a structured methodology to network troubleshooting

## Prerequisites

Students should have:

* Basic Linux command-line skills
* Familiarity with networking fundamentals (IP, DNS, ports)
* Understanding of system administration basics

## Lab Environment

* CentOS/RHEL-based Linux system
* NetworkManager enabled
* firewalld installed
* Root or sudo access
* Internet connectivity
