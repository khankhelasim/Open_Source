# Lab 9: Partitioning and Formatting Disks

## Objectives

By the end of this lab, students will be able to:

* Create and manage disk partitions using `fdisk` and `parted`
* Format partitions with various file systems using `mkfs`
* Mount and unmount file systems
* Understand MBR vs GPT partition tables
* Verify partition and file system information
* Apply proper disk management best practices

---

## Prerequisites

* Basic Linux CLI knowledge
* Understanding of file system hierarchy
* Familiarity with `sudo`
* Basic storage concepts (disk, partitions, file systems)
* Access to a Linux terminal

---

## Lab Environment Setup

* CentOS/RHEL-based Linux
* Sudo access
* Pre-attached additional disks
* All required tools installed

---

## Conclusion

* Created MBR and GPT partitions
* Formatted ext4 and xfs file systems
* Mounted and configured persistent storage
* Verified and troubleshot disk setups

These skills are essential for Linux system administration and RHCSA preparation.
