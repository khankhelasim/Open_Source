# Lab 2 Notes – File System Navigation


## Navigation Commands
- pwd → show current directory
- ls / ls -l / ls -la → list contents
- cd / cd .. / cd - / cd ~ → navigation


## File & Directory Creation
- mkdir / mkdir -p
- touch
- echo > file
- cat << EOF


## find Command
- find . -type f
- find . -type d
- find /etc -name "*.conf" -type f


## Permissions
- ls -l → view permissions
- chmod 755 file
- chmod u+x file


## Permission Values
- r = 4, w = 2, x = 1
