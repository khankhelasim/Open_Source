# Lab 2: File System Navigation (RHCSA Rapid Track)


## Overview
This lab focuses on **Linux file system navigation**, directory and file creation, organization, and **file permission management**. These skills are fundamental for **RHCSA certification** and real-world Linux administration.


## Objectives
- Navigate directories using cd, ls, pwd, and find
- Create and organize files and directories
- Understand and modify file and directory permissions
- Practice numeric and symbolic chmod usage


## Environment
- RHEL / CentOS-based Linux
- Full command-line access
- Pre-configured cloud lab environment
