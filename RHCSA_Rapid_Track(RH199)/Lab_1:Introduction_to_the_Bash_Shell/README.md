# Lab 1: Introduction to the Bash Shell (RHCSA Rapid Track)


## Overview
This lab introduces the **Bash shell**, essential Linux commands, file system navigation, input/output redirection, piping, and basic shell scripting. It is designed according to **RHCSA certification objectives**.


## Objectives
- Navigate and use the Linux command line
- Perform file and directory operations
- Use redirection and pipes
- Create and execute Bash scripts
- Build foundational Linux administration skills


## Environment
- RHEL / CentOS-based Linux
- Bash 4.2+
- Root or sudo access
## Topics Covered
- Bash shell basics
- File and directory management
- Input/output redirection
- Pipes
- Shell scripting fundamentals

## Duration
⏱️ 2–3 Hours
