By the end of this lab, students will be able to:

* Basic Linux command-line skills
* Understanding of file permissions and ownership
* Familiarity with system administration concepts
* Knowledge of log file analysis
* Access to a Red Hat-based Linux system (RHEL 9 / CentOS Stream 9)

### Lab Environment

* OS: RHEL 9 or CentOS Stream 9
* SELinux: Enabled and configured
* Required tools: Pre-installed or installed during the lab
