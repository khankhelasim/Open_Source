# Lab 10: Managing Logical Volume Management (LVM)

## Objectives
* Understand Logical Volume Management (LVM) concepts
* Create and manage Physical Volumes (PV)
* Create and manage Volume Groups (VG)
* Create, extend, and shrink Logical Volumes (LV)
* Format and mount LVM volumes
* Monitor LVM components
* Perform snapshot operations

## Prerequisites

* Basic Linux commands knowledge
* Understanding of disks and partitions
* Root or sudo access
* CentOS / RHEL-based system

## Conclusion

All LVM tasks were performed successfully. Physical volumes, volume groups, and logical volumes were created, extended, shrunk, and monitored. Snapshot functionality was tested, confirming flexible and reliable storage management using LVM.
