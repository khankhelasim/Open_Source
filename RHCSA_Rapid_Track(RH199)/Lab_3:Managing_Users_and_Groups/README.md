# Lab 3: Managing Users and Groups

This repository contains scripts and notes for practicing Linux user and group management.

## Objectives
- Create and manage user accounts
- Set and modify user passwords
- Modify existing user account details
- Create and manage groups
- Add users to groups and manage memberships
- Delete users and groups safely
- Understand `/etc/passwd`, `/etc/group`, `/etc/shadow`
- Apply security practices when managing accounts

## Usage
Run the scripts in the `commands/` folder as root or with `sudo`.
