# Lab 05: Text Processing with grep, sed, and awk

## Objective
This lab focuses on mastering Linux text-processing utilities used daily by system administrators and DevOps engineers. You will learn how to search, transform, analyze, and report data using grep, sed, and awk.

## Skills Covered

Pattern searching with grep

Text transformation using sed

Data extraction & reporting with awk

Log analysis and CSV processing

Command pipelines

## RHCSA Relevance
Text processing is a core RHCSA skill for log analysis, automation, and troubleshooting.
