## Cheat Sheets

### Nano Cheat Sheet 

```text
NANO QUICK REFERENCE
Ctrl + O  Save
Ctrl + X  Exit
Ctrl + K  Cut line
Ctrl + U  Paste
Ctrl + W  Search
Ctrl + A  Line start
Ctrl + E  Line end
Ctrl + Y  Page up
Ctrl + V  Page down
```

### vim Cheat Sheet 

```text
VIM QUICK REFERENCE

MODES
i   Insert
Esc Normal
:   Command

NAVIGATION
h j k l  Move   NAVIGATION: h,j,k,l : Left, down, up, right   
w b      Word    w : Next word b : Previous word
0 $      Line start/end  0 : Beginning of line $ : End of line
gg G     First/Last line  gg : First line G : Last line

EDITING
x  Delete char
dd Delete line
yy Copy line
p  Paste
u  Undo

SAVE / EXIT
:w   Save
:q   Quit
:wq  Save & Quit
:q!  Force Quit
```
